# System Behavior Analysis 

## Appointment Cacellation
- Cancelation will update the status of the specific appointment to cancelled.

## Doctor leaving 
- If a doctor leaves, the raw containing his details will be deleted and every other entity reference that doctor will replace him with another doctor/

## Partial payments 
- With partial payment the payment detials status will not be flagged as a complete.

## Servive price changes 
- Will update the price value inside service entity.
